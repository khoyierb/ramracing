Overview

Concept & Idea

Ram Racing is a fast-paced racing experience that offers both a challenging but entertaining time for any player.

Game Design

The game has one main track that we were able to complete within the project timeline with various 3D objects and textures.

Programming & Engine

Built in Unity, handling racing physics, scoring, and smooth control.

Launch & Updates

The project is nearly ready to be played, if it were desired we could update this game to add various elements.

Resources

Community & Forums

GameDev.net, Unity Forums offer help and collaboration. If you were interested in your own game development, we'd recommend checking these sites out as they offer great information.

Team

Development Team

Our team of computer science and business students worked together to make Ram Racing.

Roles & Responsibilities

Coders focused on game logic; others handled documentation and design.

Team Highlights

We value teamwork, creativity, and continuous improvement. Though this game was difficult to develop for various reasons we feel proud of what we were able to create.
